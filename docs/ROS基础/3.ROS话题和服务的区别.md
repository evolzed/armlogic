#### 话题和服务的区别

|          |    **话题**     |      服务       |
| :------: | :-----------: | :-----------: |
| **同步性**  |      异步       |      同步       |
| **通信模型** |     发布/订阅     |    服务器/客户端    |
| **底层协议** | ROSTCP/ROSUDP | ROSTCP/ROSUDP |
| **反馈机制** |       无       |       有       |
| **缓冲区**  |       有       |       无       |
| **实时性**  |       弱       |       强       |
| **节点关系** |      多对多      |      一对多      |
| **试用场景** |     数据传输      |     逻辑处理      |
