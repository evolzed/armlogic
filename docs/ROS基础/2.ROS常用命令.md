### ROS常用命令



|        命令        |      作用       |
| :--------------: | :-----------: |
| catin_create_pkg |     创建功能包     |
|     rospack      |   获取功能包的信息    |
|   catkin_make    |   编译工作空间的功能   |
|      rosdep      | 自动安装功能包依赖的其他包 |
|      roscd       |    功能包目录跳转    |
|      roscp       |   拷贝功能包中的文件   |
|      rosed       |   编辑功能包中的文件   |
|      rosrun      | 运行功能包中的可执行文件  |
|    roslaunch     |    运行启动文件     |
